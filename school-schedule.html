School Scedule Application

Technologies used:

Java, Android, SQLite, Room Persistence Library

School Schedule is an application that allows the user to enter school term, course, and test information. The information that can be entered includes names, dates, notes, and associated courses and tests. The information enterted will persist as data within the SQLite database. Other features include: notes sharing, and notifications.
